HTML

  • HTML is a lot like the walls of a house, it makes up the structure of the page
  • If HTML is like the walls or structure of a house, each body is like a room partitioned out by walls
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • CSS adds style to a webpage, can help make it more presentable
  • If we continue the house analogy, then CSS is a lot like the paint and curtains of a house
  • The "." in CSS is a class selector, but which the affected style changes all classes regardless of element
  • CSS elements can simply be designated before the "{}" and the html will apply said style given the proper priority
  • In order for the CSS to take effect, it needs to be linked in the html properly using both "rel:="stylesheet"" and "href="file-path""

Git

  • Git is used to submit, control and organize your work regarding well any code.
  • The code "git commit -m" should never end like that, it always will need a note or it throws a mega tantrum
  • Before ever using git push you should always use git pull origin main to ensure there are no conflicts
  • A basic git flow will go:
    • git status
    • git add -A
    • git commit -m "insert comment/summary"
    • git pull origin main
    • git push origin branchname

JS | Javascript

  • JS adds functionality, it allows for more than simple HTML interaction, it can add logic
  • Again - continuing the House analogy, we can compare JS to appliances, as to add functionality and logic
  • A variable is a named container that allows us to store data in our code
  • Control Flow is the order in which a computer executes code in a script.
  • The Control Flow typically executes from top to bottom of a command line
  • An array is a collection of stored similar data, if a function cannot define an array for its behaviour, check the command flow (execute the command after the array is established)
  • The function."name" line creates a an expected behaviour based on the logic given to it. When recalled by using "name"(); in a console, the fuction will be executed
